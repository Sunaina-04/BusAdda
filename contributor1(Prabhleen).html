Prabhleen Kaur

Role on Website: Web Developer

Aim: This project aims to create a website that provides real-time college bus arrival and departure information for students and staff, empowering them with convenient commute planning and reducing uncertainty. Beyond user convenience, the system will offer the college administration valuable data insights into route efficiency, passenger loads, and potential bottlenecks, facilitating optimized scheduling, route adjustments, and efficient resource allocation, ultimately lowering operational costs and improving service coverage. A significant aspect is the transition of security personnel from manual pen-and-paper data entry to a more efficient digital system for recording bus movements.

Contributions: I contributed to the core development of the Bus Adda project by:

  • Designing and building the Main Home Page for the platform.
  • Developing the Student Dashboard Page with multiple student-centric features.
  • Implementing the Backend Functionality for Login and Signup using Flask and MySQL.
  • Adding Export as PDF functionality for user data and forms.
  • Creating the Request a Bus Stop form to gather new bus stop suggestions.

Through this project, I enhanced my skills in both frontend and backend development and gained valuable experience in building a complete web application.
